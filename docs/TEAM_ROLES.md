# 👥 Team Roles & Responsibilities

This document outlines the areas of focus for each of the 3 team members to ensure smooth collaboration and avoid merge conflicts.

## 🟢 Member 1: Frontend Lead
**Focus:** UI/UX, Component Library, State Management.
- Responsible for the `frontend/src/pages` and `frontend/src/components` directories.
- Ensures all UI follows the **DormDesk** premium design system.
- Manages React Context and API integration on the client side.

## 🔵 Member 2: Backend & Database Lead
**Focus:** API Architecture, Database Schema, Security.
- Responsible for the `backend/controllers`, `backend/models`, and `backend/routes`.
- Ensures database integrity and optimized queries.
- Manages JWT authentication and server-side validation.

## 🟠 Member 3: Full-Stack & DevOps
**Focus:** Integration, Feature Parity, Documentation, Deployment.
- Responsible for `frontend/src/services` and root-level configurations.
- Bridges the gap between frontend requirements and backend capabilities.
- Manages the `README.md`, `CONTRIBUTING.md`, and GitHub Actions/Deployments.
